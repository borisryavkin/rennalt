# Knowledge Base Article Summaries (Evolt)

Scope: Installation & Assembly + Connection & Software Updates categories. Summaries are based on the public KB pages.

## Installation & Assembly

- Connect the Printer to the Evolt 360 Body Scanner
  - URL: https://knowledge.evolt360.com/knowledge/connect-the-printer-to-the-evolt-360-body-scanner
  - Purpose: Cable-attach Brother printer and set defaults on the kiosk.
  - Key steps: power on -> add printer -> choose Brother HL-L2300D (brlaser v4) -> set default options -> set server default -> verify default printer in Control Panel.

- Results are not aligned on the sheets when printed
  - URL: https://knowledge.evolt360.com/knowledge/results-are-not-aligned-on-the-sheets-when-printed
  - Purpose: Fix misaligned result sheets.
  - Key steps: check connection type -> prefer cable -> if wireless, delete printers and re-add correct model -> ensure same network -> adjust printer print position if needed.

- Disabling Deep Sleep on the Printer
  - URL: https://knowledge.evolt360.com/knowledge/disabling-deep-sleep-on-the-printer
  - Purpose: Stop printer from sleeping and missing prints.
  - Key steps: Menu -> General Setup -> Ecology -> Sleep Time -> set 50 minutes -> open Deep Sleep -> set OFF.

- No 'Print' button or option to 'Preview' after a scan
  - URL: https://knowledge.evolt360.com/knowledge/no-print-button-and-preview-after-scan
  - Purpose: Restore print/preview after scan.
  - Key steps: run software update -> check Operator Panel print setting Manual/Auto -> verify default printer -> re-pair printer if missing -> escalate if unresolved.

- Printer Jamming
  - URL: https://knowledge.evolt360.com/knowledge/printer-jamming
  - Purpose: Reduce repeated jams.
  - Key steps: clear jam -> fan paper -> set printer paper type to Thick -> test -> contact Brother support if persistent.

- Compatible Printers
  - URL: https://knowledge.evolt360.com/knowledge/compatible-printer-s
  - Purpose: Approved printer list.
  - Key points: Brother HL-L series compatible; HP M203W supported; Staples partnership link.

- How to connect the Evolt 360 Scanner to an HP M203W Printer
  - URL: https://knowledge.evolt360.com/knowledge/how-to-connect-evolt-360-scanner-to-hp
  - Purpose: Add HP M203W via kiosk printer menu.
  - Key steps: add printer -> select HP M203-M206 Postscript driver -> set default options -> set server default.

- Connecting the Scanner to the Brother Printer via Wi-Fi

## Connection & Software Updates

- Connecting the Evolt 360 Body Scanner to the internet
  - URL: https://knowledge.evolt360.com/knowledge/how-to-connect-to-the-network
  - Purpose: Restore internet connection.
  - Key steps: Control Panel -> Network -> Ethernet shows "100 connected" -> Wi-Fi on -> remove old profiles -> New Connection -> verify Online (green); offline scans allowed but results sync later.

- Software Update
  - URL: https://knowledge.evolt360.com/knowledge/software-update
  - Purpose: Update kiosk software.
  - Key steps: Control Panel -> Online -> Software Update -> wait 3-4 minutes and allow reboot.

- Calibrating Scanner
  - URL: https://knowledge.evolt360.com/knowledge/calibrating-scanner
  - Purpose: Fix inaccurate weight/results.
  - Key steps: Control Panel -> Scale -> set Measurement Mode Precision, Weight of Clothes 0.5kg, Scale Weight 0.00 -> Calibrate -> Save -> test Weight Only.
  - Notes: If Scale API disconnected, request photos/video before proceeding; switch to Normal mode if electrical noise.

- Enable a Passcode System for your Evolt 360 Body Scanner
  - URL: https://knowledge.evolt360.com/knowledge/i-want-to-enable-a-passcode-system-on-my-evolt-360-scanner
  - Purpose: Require passcode per scan.
  - Key steps: email support to enable -> Control Panel refresh -> faint grey circle appears -> enter operator passcode for scan credit.

- The scanner is stuck on the Control Panel screen
  - URL: https://knowledge.evolt360.com/knowledge/scanner-is-stuck-on-the-control-panel-screen
  - Purpose: Cannot start scan.
  - Key steps: ensure Online -> open Operator Panel -> enter logins from back sticker -> set Kiosk mode unattended -> save -> Start Scanning.

- No option to add 'NEW CONNECTION'
  - URL: https://knowledge.evolt360.com/knowledge/no-option-to-add-new-connection
  - Purpose: Wi-Fi list missing.
  - Key steps: confirm dongle -> System -> Lock USB OFF -> refresh -> power cycle -> test dongle -> update via Ethernet -> escalate.

- No 'Start Scanning' button
  - URL: https://knowledge.evolt360.com/knowledge/no-start-scanning-button
  - Purpose: Start button missing.
  - Key steps: power cycle -> check Scale API/Scale Status Ready -> log back into Operator Panel -> escalate if unresolved.

- Nothing happens when you tap on Log In/ Sign Up for scan
  - URL: https://knowledge.evolt360.com/knowledge/nothing-happens-when-they-try-to-select-log-in/-sign-up-for-scan
  - Purpose: Touch appears unresponsive.
  - Key steps: check faint grey circle (passcode enabled) -> enter operator passcode -> re-enter Operator Panel credentials.

- Scanner is not turning ON
  - URL: https://knowledge.evolt360.com/knowledge/scanner-is-not-turning-on
  - Purpose: Power issue.
  - Key steps: check rear switch -> reseat power cable (flat side up) -> try different outlet -> escalate if lights/sound with no boot.

- Scanner is stuck on a white screen
  - URL: https://knowledge.evolt360.com/knowledge/scanner-is-stuck-on-a-white-screen
  - Purpose: Recovery from white screen.
  - Key steps: restart -> reconnect internet -> run software update -> wait 10 minutes for recovery -> escalate.

- What is my Operator Panel log in?
  - URL: https://knowledge.evolt360.com/knowledge/what-is-my-operator-panel-log-ins
  - Purpose: Locate kiosk-specific operator login.
  - Key points: sticker on back contains email/password; contact support if missing.
